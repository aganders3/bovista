MinipVolume
Minimum Intensity Projection.
MipVolume
Maximum Intensity Projection. With attenuation == 0 this is plain MIP; with attenuation > 0 each sample is scaled by exp(-attenuation * sumval) where sumval is the running normalised density along the ray — weighting near-camera regions more heavily (“attenuated MIP” in napari).
VolumeCore
Shared GPU state for every volume rendering visual. Each concrete public type (DirectVolume, MipVolume, IsosurfaceVolume, …) owns one of these plus its mode-specific parameters. The pipeline inside the core is fixed at construction to the right mode’s fragment entry point.
