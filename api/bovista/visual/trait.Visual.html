Base trait for all visual objects that can be rendered in a scene

Source

fn prepare(&mut self, device: &Device, queue: &Queue, camera_info: &CameraInfo)

Prepare the visual for rendering (called once per frame before render) This is where uniform buffers should be updated, etc. Camera info is provided for visuals that need LOD selection or culling

Source

fn render(&self, render_pass: &mut RenderPass<'_>)

Render the visual to the given render pass Note: This method cannot take ’self by reference with a lifetime because it needs to work with Mutex guards in the scene graph
