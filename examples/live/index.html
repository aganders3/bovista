🍄 Bovista Examples

High-performance 3D visualization with WebGPU

Slice Viewer

Load multiscale OME-Zarr datasets from remote S3 and explore them with an interactive slice plane.

  • On-demand tile streaming from S3
  • Multiscale pyramid (LOD) support
  • Multi-channel fluorescence imaging
  • Arbitrary slice plane: tilt, rotate, offset
  • Orthographic + perspective projection
  • Real-time window/level contrast
Launch Demo →

Volume Renderer

Direct volume rendering via GPU ray marching. Same tile-streaming backend as the slice viewer — no extra data required.

  • Front-to-back alpha-composited ray march
  • Adjustable step size and density scale
  • Same virtual-texture tile cache as slice viewer
  • Multiscale LOD streaming from remote S3
  • Real-time window/level contrast
  • Debug LOD tinting
Launch Demo →

System Requirements

  • Modern browser with WebGPU support (Chrome 113+, Edge 113+, Safari Technology Preview)
  • WebGPU-capable GPU (most modern GPUs from 2018+)
  • Check compatibility: webgpureport.org
